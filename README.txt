
Joda-Money
=========
Joda-Money provides essential currency and money classes to supplement the JDK.

See the home page for more details:
http://joda-money.sourceforge.net/

See the user guide:
http://joda-money.sourceforge.net/userguide.html

The source code and issue tracking are at GitHub:
https://github.com/JodaOrg/joda-money

The mailing list and downloads are at Sourceforge:
https://sourceforge.net/projects/joda-money/
